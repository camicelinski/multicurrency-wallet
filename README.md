### Multicurrency wallet

The goal of this project was to create a currency wallet app that allows user to store data about currencies recently bought.

&nbsp;

## :bulb: Features

- add and remove currencies
- profit/loss calculation
- currency exchange rate provided by [exchangeAPI](https://exchangeratesapi.io/)
- based on the selected purchase date, the app will suggest the exchange rate for the day
- data is stored automatically to Local Storage

## ‚öôÔ∏è Installation

The project uses [node](https://nodejs.org/en/), [npm](https://www.npmjs.com/). Follow the steps below to run it locally. 

- Clone the project using:

```bash
  git clone
```

- Go to the project directory and install all dependencies:

```bash
  npm i
```

- To run webpack and create a localhost in developer mode type:

```bash
  npm start
```
